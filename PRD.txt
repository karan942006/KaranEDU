# Recommended Project (Best for Students)

## Project Name

**EduAI – Personalized Learning Assistant**

### Why This SDG?

Millions of students struggle to access personalized learning support. Many cannot afford private tutors and often get stuck while studying.

---

# Complete PRD (Product Requirement Document)

## 1. Project Overview

### Project Name

EduAI – Personalized Learning Assistant

### SDG

SDG 4: Quality Education

### Problem Statement

Many students face difficulties in understanding concepts, completing assignments, and receiving personalized guidance. Traditional classroom teaching cannot always provide individual attention to every student.

As a result:

* Students lose interest in learning.
* Academic performance decreases.
* Doubts remain unresolved.
* Learning becomes inefficient.

### Objective

Develop an AI-powered educational assistant that:

* Answers student questions.
* Explains concepts in simple language.
* Generates quizzes.
* Recommends study resources.
* Provides personalized learning support.

# 2. Target Users

### Primary Users

* School Students
* College Students
* Competitive Exam Aspirants

### Secondary Users

* Teachers
* Educational Institutions

---

# 3. Solution Description

EduAI acts as a virtual tutor available 24/7.

The user enters a question such as:

> "Explain Artificial Intelligence"

The AI:

* Understands the query
* Generates a simple explanation
* Suggests resources
* Creates practice questions

This makes learning interactive and personalized.

---

# 4. Core Features

## Feature 1: AI Chat Tutor

### Description

Students can ask doubts in natural language.

### Example

User:

> What is Machine Learning?

AI:

> Machine Learning is a branch of AI that enables computers to learn from data without explicit programming.

---

## Feature 2: Quiz Generator

### Description

Automatically generates quizzes from any topic.

### Example

Topic:

> Artificial Intelligence

Generated Quiz:

1. What is AI?
2. What is Machine Learning?
3. What is NLP?

---

## Feature 3: Resource Recommendation

### Description

Suggests:

* YouTube videos
* Articles
* Notes
* Books

Based on student's query.

---

## Feature 4: Learning Dashboard

### Description

Shows:

* Questions asked
* Topics learned
* Quiz scores
* Learning progress

---

## Feature 5: AI Study Planner

### Description

Creates personalized study schedules.

Example:

| Time     | Subject  |
| -------- | -------- |
| 9-10 AM  | AI       |
| 10-11 AM | DBMS     |
| 11-12 PM | Aptitude |

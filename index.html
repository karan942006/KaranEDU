Core Capabilities

Everything a Student
Needs to Excel

Five intelligent modules working together — powered by Gemini AI, cached offline by Room SQLite, and synced securely to Firebase when connectivity returns.

✨

Personal AI Tutor (Gemini Powered)

Real-time conversational answers to any academic question. Grade-level aligned responses, concept breakdowns, mnemonics, and detailed explanations — all powered by Google Gemini AI and cached locally for offline access.

Google Gemini · Room Cache
🛡

AI Diagnostic Quizzes

Automatically generated assessments tailored to your subject and grade. Instant performance breakdown, detailed explanations per question, and diagnostic scorecards to identify exactly where to improve.

Auto-Generated · Instant Feedback
🃏

Spaced-Repetition Flashcards

AI creates bite-sized flashcards from any topic. Interactive 3D flip cards for active recall, with Easy/Medium/Hard self-rating to optimize your spaced-repetition schedule automatically.

Active Recall · AI Generated
📅

Smart Daily Study Planner

Custom tracks for Mathematics, Physics, Essay prep and more. Interactive task checklists that persist locally — your study progress survives app restarts, offline sessions, and device changes.

Local Persistence · Custom Tracks
☁

SQLite Cache & Cloud Sync

Local Room database stores all quiz results, chat history, flashcards, and planner data indefinitely offline. When connectivity returns, Firebase Firestore seamlessly syncs everything across devices.

Room DB · Firebase Sync

Architecture & Tech Stack

Built on Modern
Android Standards

MVVM architecture with clean separation of concerns — from Jetpack Compose UI down to Room SQLite and Firebase Firestore.

🎨

Kotlin & Jetpack Compose (M3)

Fully declarative UI with Material Design 3 layouts. Edge-to-edge support, adaptive screen sizing, and rich light/dark theming across all Android OEM skins.

com.karanedu.app.ui.screens
🧠

KaranEDU ViewModel (MVVM)

Lifecycle-aware state management using Kotlin Coroutines and StateFlow. Single source of truth for AI responses, quiz state, flashcard progress, and planner data.

StateFlow · Coroutines · lifecycle-aware
🗄

Room SQLite — Offline-First Cache

All chat history, quiz scores, flashcards, and study plans stored in Room DB. The app works fully without internet — offline is the primary mode, not a fallback.

com.karanedu.app.data.database
🤖

Google Gemini SDK Integration

Async API calls via Kotlin Coroutines for chat tutoring, quiz generation, and flashcard creation. Responses cached locally for instant offline replay.

Gemini AI · Coroutines · async
🔥

Firebase Auth & Cloud Firestore

Gmail-based onboarding with secure OAuth. Firestore backup and restore syncs all student progress, quiz history, and study plans across multiple devices seamlessly.

Firebase Auth · Firestore · Cloud Sync
